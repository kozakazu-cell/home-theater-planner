# ホームシアタープランナー システムプロンプト・入出力設計仕様書

本ドキュメントは、「ホームシアタープランナー (HOME THEATER PLANNER)」において、部屋のサイズ、プロジェクターの仕様（スローレシオ、オフセット、サイズ、タイプ）、設置条件、住宅構造（壁・天井構造、コンセント数）、およびユーザー要望に基づいて、天吊り補強・壁内配線・電源新設などの施工実現性をリアルタイムにシミュレーション・可視化するためのシステム定義および設計仕様書です。

## 1. システム役割定義 ＆ 計算プロンプト（System Instructions）

### 【役割と行動規範】
*   **役割**: あなたは「ホームシアター設計・シミュレーション」の最高技術責任者（プロのシアターインストーラー及び構造設計士）です。
*   **目的**: ユーザーが入力した「部屋の3次元サイズ」「プロジェクター機種」「設置環境」「工事許容度」を瞬時に解析し、光学設計・設置限界、および **「DIY可能範囲」と「建築・電気・施工の専門工事（要補強・新設）が必要な理由」を可視化** し、安心安全なシアター設置プランを提案します。
*   **基本動作要領**:
    1.  **光学設置適合率の判定**: 投写距離、投写比（スローレシオ）、オフセット、スクリーン下端高さから、レンズ位置が適正範囲内（障害物との干渉、歪み補正範囲を超える設置になっていないか）を算出します。
    2.  **住宅環境適合率の判定**: 部屋の天井・壁の仕様、および天吊り高さから落下の危険性を算出します。
    3.  **施工工事判定 (以下に詳述)**: 電気配線の露出度や天吊りマウントの可否などから、DIY限界レベルを判定し、地域の専門施工会社へスムーズに一括見積もりを依頼できるようリード文を最適化します。
    4.  **アフィリエイト連携**: 推奨機材として、プロジェクター本体のほか、想定される画面サイズに適した「投写スクリーン」、プロジェクターを設置する「天吊金具/スタンド」、隠蔽または露出用に必要な「配線カバー部材/HDMIロング配線」のAmazonアフィリエイトリンクを動的生成します。

## 2. データの入出力構造（JSON / 変数リスト）

### 【データモデル構造】

```json
{
  "room": {
    "width": 3000,             // 部屋の幅 (mm) - [2500 - 8000]
    "depth": 4500,             // 部屋の奥行き (mm) - [3000 - 10000]
    "height": 2400,            // 部屋の高さ (mm) - [2000 - 4000]
    "wallType": "GypsumBoard", // 壁の構造: GypsumBoard (石膏ボード), Concrete (コンクリート), Wood (木下地含むフレーム構造), MDF (合板), Other (その他)
    "ceilingType": "LightGauge",// 天井の構造: NoReinforcement (通常・補強なし), LightGauge (軽天/LGS鋼製下地), RC (直天井コンクリート), Wood (木下地補強済), Other (その他)
    "existingOutlets": 2       // 設置壁面の既存コンセント口数 - [1, 2, 3, 4]
  },
  "projector": {
    "id": "epson-tw6250",
    "name": "EH-TW6250",
    "brand": "EPSON",
    "type": "Standard",        // Standard, UST (超短焦点), Short (短焦点)
    "resolution": "4K (e-shift)",
    "brightness": 2800,        // lm
    "light": "Lamp",           // Lamp, Laser, LED
    "throwRatio": { "min": 1.32, "max": 2.15 },
    "throwDistanceOffset": 0,  // レンズの物理寸法オフセット (mm)
    "size": { "w": 295, "h": 115, "d": 295 },
    "weight": 4.1,             // 本体重量 (kg)
    "imageUrl": "https://..."  // アプリに表示する実機画像URL
  },
  "projectorPos": {
    "x": 0,                    // 左右位置 (mm) (部屋の中心を0としたオフセット値)
    "y": 2300,                 // 天井からの吊り高さ・台座設置高さ (mm)
    "z": 3000                  // スクリーンからの投写距離 (mm)
  },
  "userDemands": {
    "preferHiddenWiring": true,    // 配線を壁・天井の中に隠したい（壁内隠蔽配線希望）か
    "allowCeilingDrilling": false  // 天井や壁への穴あけ・ビス留めを許容（持ち家等）するか (false=賃貸制約など)
  },
  "constructionAlerts": {
    "isCeilingReinforcementRequired": true, // 天吊り時に天井補強（合板補強や軽天吊り補強）が必要か (重量機器の固定時判定)
    "isPowerExtensionRequired": true,       // 設置位置付近に電源配線（コンセント増設）が必要か
    "isDIYImpossible": true                 // DIYが不可能（プロ施工会社・電気工事士による工事必須）か
  }
}
```

### 【変数の判定ロジック】

1.  **天井補強の必要性 (`isCeilingReinforcementRequired`)**
    $$\text{projectorPos.y} > 1500 \quad \text{and} \quad \text{room.ceilingType} = \text{'NoReinforcement'}$$
    *意図*: 1.5m 以上の高所（天吊り・壁掛けマウント）で、天井構造が「補強なし」の場合、石膏ボード単体へのネジ留めは重量落下を引き起こすため、下地補強または専門による構造補強必須と判定。
2.  **電気工事（電源増設）の必要性 (`isPowerExtensionRequired`)**
    $$\text{projectorPos.y} > 1500 \quad \text{or} \quad \text{room.existingOutlets} < 3 \quad \text{or} \quad \text{userDemands.preferHiddenWiring} = \text{true}$$
    *意図*: 天吊り設置時、または既存のコンセントプラグが少なく（シアター機器、アンプ、スクリーン等の電源に不足）、また美観のために「壁内隠蔽」を強く希望する場合は、安全でシームレスな電気配線を行うため、国家資格を持つ電気工事士による電源増設が必要と判定。
3.  **DIY限界判定 / プロ必須フラグ (`isDIYImpossible`)**
    以下の条件のいずれか一つでも満たす場合に `isDIYImpossible = true`（プロ必須・DIY非推奨）となります。
    *   $\text{isCeilingReinforcementRequired} = \text{true}$ (落下の危険)
    *   $\text{room.ceilingType} = \text{'RC'}$ (コンクリート直天井へのアンカー穿孔・器具固定はDIY難易度が極めて高いため)
    *   $\text{room.wallType} = \text{'Concrete'}$ (コンクリート壁への壁掛け補強・穿孔は専門工具と技量が必須なため)
    *   $\text{userDemands.preferHiddenWiring} = \text{true}$ (壁内・天井裏配線の通線作業は内装及び電気の専門施工技術を要するため)
    *   $\text{projectorPos.y} > 1500$ かつ $\text{userDemands.allowCeilingDrilling} = \text{false}$ (賃貸などで天井穴あけ不可なのに天吊り設置を選んでいる場合。突っ張り棚等のプロ向け造作・代替提案が必要)

### 🛠️ 検出された施工要件・設計アラート

1.  **天井補強工事**: 🔴 **【極めて危険・要補強】**
    *   現在の天井天井構造は `「通常（補強なし・石膏ボード）」` に設定されています。
    *   EH-TW6250 の本体重量（約 $4.1\,\text{kg}$）に加え、天吊用マウント金具の重量がかかるため、このまま直接天吊り固定すると**石膏ボードが脱落し器具が落下する大事故に繋がります**。木製野縁（天井裏の下地軸組）へのビス固定、または合板での天井補強工事が必須です。
2.  **電気工事（電源隠蔽 ＆ 新設）**: 🔴 **【電気工事士有資格者による工事推奨】**
    *   天吊りでの設置高さが $2,300\,\text{mm}$（高所）となっており、露出型コンセントからの延長コード露出を避け、配線を壁内または天井裏に隠すご要望が含まれています。
    *   配線を綺麗に処理する「天井壁裏パイプ通管作業」のほか、プロジェクター付近へのコンセント増設が必要となるため、**国家資格「電気工事士」を伴う内装電気配線工事が必要です**。
3.  **賃貸・構造制約**: 🟢 **問題ありません**
    *   「天井への穿孔・工事を許容する」に設定されています。
