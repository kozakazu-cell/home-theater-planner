# ホームシアタープランナー システム定義書 (System Instructions)

## 1. アプリケーションの役割定義 (System Instructions)
本アプリケーション「HOME THEATER PLANNER」は、ユーザーが指定した部屋の寸法に基づいて、プロジェクターの最適な配置、スクリーンサイズ、およびアフィリエイト（購買）への導線を視覚的かつ論理的にシミュレーションするツールです。

**主な機能と計算ロジック:**
- **投写距離の計算:** 選択されたプロジェクターの「スローレシオ (Throw Ratio)」と「スクリーン幅」に基づき、必要なレンズから壁までの距離幅（最小〜最大投写距離）を算定。
- **オフセットの計算:** 「打ち込み角（Offset Percent）」に基づき、レンズ中心から結像するスクリーンの下端（または上端）への垂直距離を計算し、製品とスクリーンの高さを干渉なく配置可能か判定。
- **設置判定:** プロジェクターの物理サイズ（奥行き）や「壁からの必須離隔距離（Throw Distance Offset）」を加味し、部屋の寸法（奥行き・高さ）に収まるかを自動判定。
- **最適な視聴距離:** 画面サイズ（インチ）に対して、`1.0倍〜1.5倍` （視野角基準）などを視聴位置・推奨配置としてサジェスト。
- **機材購入のコンテキスト生成:** 計算結果とプロジェクターの機種に基づき、最適なスクリーンサイズや天吊り金具/自立スタンド、eARC対応サウンドバーなど、セットアップに必要な音響・周辺機材のAmazon検索導線を自動生成する。

## 2. データの入出力構造（JSON / 変数リスト）

本システムはReactにおけるState（状態）としてデータを保持し、リアルタイムにシミュレーションへ反映します。

### [INPUT] ユーザー操作・環境変数 (TheaterState)
```json
{
  "room": {
    "width": 3000,    // 部屋の幅 (mm)
    "height": 2400,   // 部屋の高さ (mm)
    "depth": 3000     // 部屋の奥行き (mm)
  },
  "projector": {
    "id": "1",
    "brand": "Anker",
    "name": "Nebula Capsule 3 Laser",
    "type": "Mobile",      // UST(超短焦点), Short(短焦点), Standard(標準), Mobile
    "throwRatio": { "min": 1.2, "max": 1.2 }, // スローレシオ
    "offsetPercent": 100,  // オフセット（垂直打ち込み角 %）
    "size": { "w": 83, "h": 170, "d": 83 }, // 機材サイズ (mm)
    "throwDistanceOffset": 0 // 必須の壁から離す距離補正 (mm)
  },
  "projectorPos": {
    "x": 0,       // 横方向位置 (mm, 中央が0)
    "y": 1000,    // 高さ位置 (mm)
    "z": 1800     // レンズからスクリーンまでの距離 (mm)
  },
  "screenSizeInch": 100,  // 想定スクリーンサイズ (インチ)
  "screenBottomY": 600,   // スクリーン下端の高さ (mm)
  "audiencePos": {
    "x": 0,
    "y": 0,
    "z": 2400     // 視聴位置（ソファ等）の距離 (mm)
  }
}
```

### [OUTPUT] アプリが計算する内部パラメーター
```json
{
  "screenSz": {
    "w": 2213.9,  // スクリーン幅 (mm) -> screenSizeInch * 25.4 * cos(atan(9/16))
    "h": 1245.3   // スクリーン高さ (mm) -> screenSizeInch * 25.4 * sin(atan(9/16))
  },
  "optimalDistances": {
    "min": 2540,  // 最短視聴距離 (mm) -> screenSizeInch * 25.4 * 1.0
    "max": 3810   // 最長視聴距離 (mm) -> screenSizeInch * 25.4 * 1.5
  },
  "throwRange": {
    "minTargetZ": 2656, // 指定画面サイズでの最短投写距離 (mm) -> screenSz.w * projector.throwRatio.min
    "maxTargetZ": 2656  // 指定画面サイズでの最長投写距離 (mm) -> screenSz.w * projector.throwRatio.max
  },
  "isInstallationValid": false // projectorPos.z が minTargetZ と maxTargetZ の範囲内にあるか判定
}
```

### 投影スペック (正常時)
**干渉なし / OK**
